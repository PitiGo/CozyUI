## Overview
A comprehensive audit of the CozyUI codebase was conducted to identify performance bottlenecks, memory leaks, and code quality issues. Several critical and high-priority improvements were implemented.

## đź”´ Critical Fixes
1.  **Memory Leak in `MaskEditor.jsx`**
    *   **Problem:** Every brush stroke created a new `blob:` URL using `URL.createObjectURL` without ever revoking the previous one. This caused rapid memory growth during masking.
    *   **Fix:** Implemented `lastBlobUrlRef` to track and `URL.revokeObjectURL()` old blobs before creating new ones. Added cleanup on component unmount. Added throttling to mask updates.

2.  **Code Duplication & DRY Violation**
    *   **Problem:** The `getEdgeColor` function was duplicated in `App.jsx` and `useWorkflow.jsx`.
    *   **Fix:** Centralized `getEdgeColor` in `useWorkflow.jsx` and exported it for use in `App.jsx`.

3.  **Unnecessary GPU Resource Usage**
    *   **Problem:** `useWebGPU` hook was calling `adapter.requestDevice()` solely to check for support. This is heavy and reserves a GPU context unnecessarily.
    *   **Fix:** Removed `requestDevice()`. We now rely on `requestAdapter()` and `requestAdapterInfo()`, which is sufficient for detection.

## đźźˇ Important Improvements
4.  **Performance Optimization (Store)**
    *   **Problem:** The `useStore` context value was not memoized, causing the entire application to re-render on every state update, even if the data didn't change for a specific component.
    *   **Fix:** Wrapped the context value in `useMemo`.

5.  **Broken Dynamic CSS**
    *   **Problem:** `BaseNode.jsx` used dynamic Tailwind classes like `min-w-[${minWidth}px]`, which fails at build time because Tailwind cannot purge unknown classes.
    *   **Fix:** Replaced with inline `style={{ minWidth }}`.

6.  **Missing Edge Colors**
    *   **Problem:** Edges originating from `InpaintingNode` ('inpaint-out') and `BackgroundRemovalNode` ('bg-removed-out') had no color defined, defaulting to "unknown".
    *   **Fix:** Added specific colors (violet and rose) to the `getEdgeColor` mapping.

7.  **UX: Toast Accumulation**
    *   **Problem:** Toast notifications could pile up indefinitely.
    *   **Fix:** Implemented a `MAX_TOASTS = 5` limit in `useToast.jsx`, auto-dismissing older toasts.

8.  **UX: Keyboard Shortcuts**
    *   **Problem:** The `Ctrl+R` shortcut (Reset Canvas) aggressively overrode the browser's native specific refresh command.
    *   **Fix:** Changed shortcut to `Ctrl+Shift+R` and updated the Sidebar documentation.

9.  **SEO & Meta Tags**
    *   **Problem:** `index.html` lacked description and OpenGraph tags.
    *   **Fix:** Added proper meta description, OG title/description, and theme color.
